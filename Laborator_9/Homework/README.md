## Description
The project is a music database management application. Create, read, update and delete functionalities for genres, artists and albums are implemented within the application.

## Installation
To run the application, you need to download the project from GitHub and open the Main.java file in a code editor. Make sure you have the appropriate Java version installed on your computer. Next, run the Main.java file.

## Use
When the application is opened, a new database is created. Next, the user can add genres, artists and albums, as well as read, update and delete existing records in the database. To perform these operations, the user can use the methods implemented in the GenreRepository, ArtistRepository, and AlbumRepository classes.
